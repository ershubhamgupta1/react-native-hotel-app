Download this beautiful free React Native starter kit, featuring a items app, to bootstrap your mobile app development. Learn React Native by working on a real project. Get familiar with various native components, navigation, redux and more.

## Getting Started

To run the app, simply run

```yarn install && expo start```

And scan the QR code in the Expo client app.

## Make Android APK

command 1: react-native bundle --platform android --dev false --entry-file index.js --bundle-output android/app/src/main/assets/index.android.bundle --assets-dest android/app/src/main/res


command 2: eas build -p android --profile preview
